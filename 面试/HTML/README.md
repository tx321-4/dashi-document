### <a name="chapter-two-four" id="chapter-two-four"></a>2.4 meta标签
> [返回目录](#chapter-one)

* meta 辅助型标签，位于head标签中
* meta 可以被使用浏览器、搜索引擎、web服务调用
* charset 定义文档的字符编码
* content 定义 http-equiv 或者name 属性相关的元信息
